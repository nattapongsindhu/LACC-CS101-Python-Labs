# LACC-CS101-Python-Labs

Python lab assignments for CS101 at Los Angeles City College — covering fundamentals, data structures, and scripting.

## Topics Covered
- Variables, data types, and expressions
- Control structures (if/else, loops)
- Functions and modules
- String manipulation
- File I/O
- Basic data structures (lists, dictionaries)
